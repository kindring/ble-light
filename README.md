# 蓝牙补光灯
> [!tip] 该项目基于安信可的蓝牙模块 pb02 开发,  
> 使用 pwm 控制四路 led 灯, 分为两组, 一组为冷光, 一组为暖光  
> 可以通过蓝牙控制灯的亮度, 色温, 风扇速度, 模式切换, 单路调节, 频率调节等功能

## 功能介绍

1. 亮度调节
2. 色温调节
3. 主动风扇散热
4. 电量监控(基于小程序问题, 暂未集成为指令功能)
